Представленный код направлен на тестирование интернет-магазина "Читай-город"
Тестирование затрагивает методы UI и API
API тестирование предусматривает как пзитивные, так и негативные сценарии
в коде используются такие дополнительные расширения как:
    pytest
    allure
    reqest
    selenium
    pytest-xdist

Выполнение тестирования и просмотр отчета в allure:
    1. Выполнить команду ./run.bat + ui или api или all (выбираем какой вид теста сделать)
    2. Дождаться выполнения тестов
    3. Выполнить команду allure open final-report
    4. Иизучть отчет
    5. Завершить просмотр отчета:
       1. Нажать сочетание клавиш Ctrl+C
       2. Подтвердить выход вводом буквы "Y"
