Представленный код направлен на тестирование интернет-магазина "Читай-город"
Тестирование затрагивает методы UI и API
    UI тесты - проверка пользовательского интерфейса (главная страница, корзина, каталог, карточка товара)
    API тесты - проверка бэкенда (корзина, закладки, авторизация)
        API тестирование предусматривает как позитивные, так и негативные сценарии

В коде используются такие дополнительные расширения как:
- `pytest` - фреймворк для тестирования
- `allure` - генерация отчетов
- `requests` - API тестирование
- `selenium` - UI тестирование

Структура проекта
chitay_gorod/
├── test/
│ ├── api/
│ │ └── test_api.py # API тесты
│ └── ui/
│ ├── test_cart.py # UI тесты корзины
│ ├── test_cataloge.py # UI тесты каталога
│ ├── test_main.py # UI тесты главной страницы
│ └── test_product.py # UI тесты карточки товара
├── Page.py # Page Object класс
├── conftest.py # Фикстуры pytest
├── config.py # Настройки (URL, токен)
├── config.py.example # Шаблон конфигурации
├── pytest.ini # Конфигурация pytest с маркерами
├── requirements.txt # Зависимости
├── run.bat # Скрипт запуска
├── .gitignore # Игнорируемые файлы
└── README.md # Документация

    1. Клонировать репозиторий:
        git clone https://github.com/Yarch120/chitay_gorod
        cd chitay-gorod
    2. Установить зависимости:
         ```bash
        pip install -r requirements.txt
    3. Установка Allure:
        scoop install allure
    4. Настройка конфигурации
        cp config.py.example config.py (Токен авторизации будет прислан в сообщении)

Запуск тестов:
    1. Через pytest:
        pytest -m "ui"  # Запуск только UI тестов
        pytest -m "api"  # Запуск только API тестов
        pytest  # Запуск всех тестов
    
    2. Через run.bat(с Allure отчетом):
        run.bat ui  # Запуск только UI тестов
        run.bat api  # Запуск только API тестов
        run.bat all  # Запуск всех тестов

Просмотр отчета:

    1.После выполнения тестов через run.bat отчет откроется автоматически
        (Если нужно открыть вручную в терминале используем команду "allure open final-report")
    2.Изучить отчет
    3.Завершить просмотр отчета:
        1.Нажать сочетание клавиш Ctrl+C
        2.Подтвердить выход вводом буквы Y

Требования
Python 3.8+
Google Chrome браузер
ChromeDriver (устанавливается автоматически через Selenium)
